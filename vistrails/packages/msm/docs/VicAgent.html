VicAgent

 creates a full simulation of the Variable Infiltration Capacity (VIC) model.
VIC is a macroscale hydrologic model that solves full water and energy balances.

If the user only wants to perform a water balance VIC simulation, only 4 forcing inputs need to
be included: [Minimum Temperature, Maximum Temperature, Total Precipitation, Wind Speed].
However, if full water and energy balance VIC simulation is required, the user should include 7
forcing inputs: [Air Temperature, Short-wave radiation, Long-wave radiation, Precipitation, Pressure,
Water-vapor pressure, and Wind Speed].
The VIC Agent requires that all forcing inputs display the same spatial extent and temporal resolution.
Full VIC documentation available at [https://vic.readthedocs.io/en/master/].

[INPUTS]
AIR_TEMP: [C] Air Temperature.
LWDOWN: [W/m^2] Long-wave downwards radiation.
SWDOWN: [W/m^2] Short-wave downwards radiation.
PREC: [mm] Total Precipitation per time-step.
PRESSURE: [kPa] Atmospheric Pressure.
TMAX: [C] Maximum Temperature measured during each time-step.
TMIN: [C] Minimum Temperature measured during each time-step.
VP: [kPa] Water-vapor pressure.
WIND: [m/s] Total magnitude of Wind Speed.
working_directory: [Optional] Folder where the VIC simulation files will be created.
The default folder will be at ['.\vistrails\packages\msm\utils\uploaded\user_model_agents\VicAgent\tmp'].

[OUTPUTS]
For a complete list of the available outputs, please visit [https://vic.readthedocs.io/en/master/
Documentation/Drivers/Classic/FluxOutputFiles/]
