DHSVMAgent

 creates a full simulation of the Distributed Hydrology Soil Vegetation Model.
A model that \"numerically represents with high spatial resolution the effects of local weather, topography,
soil type, and vegetation on hydrologic processes within watersheds.\" (Pacific Northwest Lab, 2019).
Documentation available at (https://dhsvm.pnnl.gov/index.stm)

This version of the DHSVMAgent automatically creates the input files necessary to run the DHSVM. However,
it performs approximations to compute the sizes of the channels, as well as uses default values for
soil and vegetation properties. The user is free to use the generated files and modify them to fit their
specific simulation.

[INPUTS]
Pressure: [Pa] Atmospheric Pressure.
Precipitation: [m] Total Precipitation per time-step.
LongWave: [W/m2] Long-wave downwards radiation.
ShortWave: [W/m2] Short-wave downwards radiation.
SpecificHumidity: [kg/kg] Specific humidity.
Temperature: [C] Ait Temperature.
Wind: [m/s] Minimum Temperature measured during each time-step.

DEM map: Path to the DEM map file.
vegetation map: Path to the vegetation map file.
soil map: Path to the soil map file.
soilDepth map: Path to the soilDepth map file.
outputCoordinates: Latitude, longitude coordinates of the outlet of the basin in the DEM provided.
resolution (m): Target resolution for the model. The closer this number is to the real resolution, the more unstable the model gets.
basinThreshold: Number of cells in the DEM map that require to be accumulated before they turn into a strem.
outputFlow (m3/s): Flow of the channel at the outlet of the basin. This is used to dimension all the channels in proportion to this one.
outputWitdh (m): Width of the channel at the outlet of the basin. This is used to dimension all the channels in proportion to this one.
minSlope: Minimum slope allowed in the drainage network.
manning: Manning rugosity parameter used for all the streams.
directory: [Optional] Folder where the VIC simulation files will be created.
The default folder will lie at ['.\vistrails\packages\msm\utils\uploaded\user_model_agents\DHSVMAgent\tmp'].

[OUTPUTS]
Streamflow: [m3/timestep] Time series with the streamflow per simulated timestep at the outlet.
