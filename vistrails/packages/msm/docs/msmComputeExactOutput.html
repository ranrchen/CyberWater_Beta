msmComputeExactOutput

 Finds the exact location of the outlet of a basin, given a DEM.
This module tries to build a basin with an area given as input. Then locates the best output cell.

[INPUTS]
01_space_range: The SpaceRange where the DEM is located
DEM_file: The path of the DEM file (ASCII).
basin_area(km2): Approximate area of the basin
estimated_coordinate(lon,lat): Best guess of where the outlet is located.

[OUTPUTS]
exact_output_coord: The exact location of the outlet in the DEM. (lon,lat)
