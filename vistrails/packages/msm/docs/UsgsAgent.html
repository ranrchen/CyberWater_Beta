UsgsAgent

 access the Water Data for the Nation databse of the US Geological Survey (USGS).
Downloads the variable selected, available on all the stations inside the spatial extent defined.
Daily information can be manually accessed at [https://waterdata.usgs.gov/nwis/dv?referred_
module=sw&search_criteria=lat_long_bounding_box&submitted_form=introduction]
Instantaneous information can be manually accessed at [https://waterdata.usgs.gov/nwis/uv/?referred_module=sw]

[INPUTS]
case_study: Tuple of TimeRange-SpaceRange defining the spatio-temporal extent of the simulation.
url: The default USGS URL used to download the values. The one that ends in 'dv/?' is for daily-values
whereas the one that ends in 'iv/?' is for instantaneous values (station-dependent time resolution).
variable_code: The code of the variable you want to download. Default = 00060 - Streamflow [ft3/s]. Full catalogue
available filling the form at: [https://nwis.waterdata.usgs.gov/usa/nwis/pmcodes].
timeout: Number of seconds we will wait for the server to reply.
to_meters: Factor by which to multiply the values to obtain meters.
keep_original_timeseries: Leave the time-series as it is?.
force_finer_scale: Force the Data Agent to use a finer time-resolution?
remove_gauges_random: Randomly remove gauges?
remove_gauges_msm: Experimental variable.
remove_gauges_amount: Removes the number of stations specified.
amount_gauges_used: Fixes the number of stations used.

[OUTPUTS]
dataset_name: Name of the resulting downloaded dataset.
Documentation at [https://hydro1.gesdisc.eosdis.nasa.gov/data/NLDAS/README.NLDAS2.pdf]
