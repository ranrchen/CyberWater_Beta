FileToDataset

 Creates a single dataset from a single file containing a timeseries .
The file must have a single column with no headers, and contain only numerical values

[INPUTS]
01_case_study: Set of TimeRange-Space range for your simulation.
02_variable_name: Name of the variable that will be stored in the dataset. i.e. precipitation, temperature, etc..
(this value is only used as label and does not play a role in the calculations.)
03_filename: File containing the column of values.
04_units: [Optional] Units of the dataset that will be uploaded. This value will be used only as a label.
05_column_index: [Optional] The column that will be uploaded as the new Dataset. Default = 0.
06_empty_symbol: [Optional] String representation of empty (NULL, None) values.
07_ready?: [Optional] True/False flag used if the user wants to time the execution of this module after another module is done.

[OUTPUTS]
out_dataset_name: The name of the resulting dataset.
