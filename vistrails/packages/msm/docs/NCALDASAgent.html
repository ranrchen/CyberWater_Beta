NCALDASAgent

 access the Earth Data services of NASA.
NCALDAS model offers various hydro-meteorological variables in a daily basis, 1/8 degree resolution.
You need to create an account on Earth Data following the link:
https://urs.earthdata.nasa.gov/

Projection: Latitude, Longitude
Spatial Coverage: -125.0,25.0,-67.0,53.0(W,N,E,S)
Temporal Coverage: 1979-01-02 to 2016-12-31
Spatial Resolution: 0.125 x 0.125 degrees
Temporal Resolution: Daily

[INPUTS]
case_study: Tuple of TimeRange-SpaceRange defining the spatio-temporal extent of the download.
The information downloaded will always fall inside these limits, in case the spatial information, or temporal
divisions do not fall exaclty at the limits set by the user in his or her 'studycase'.
password: Your Earth Data password credentials.
username: Your Earth Data username credentials.
rename_variable: [Optional] Input used if you want to rename the variable name of this dataset.
variableName: The variable you want to download (including units).
Possible options:
    Baseflow-groundwater runoff: [mm/s]
    Storm surface runoff: [mm/s]
    Pressure [Pa]
    Evapotranspiration: [mm/s]
    Temperature: [K]
    Temperature Min: [K]
    Temperature Max: [K]
    Total Precipitation Rate: [mm/s]
    Wind speed: [m/s]
    Soil temperature 0-10cm: [K]
    Soil temperature 10-40cm: [K]
    Soil temperature 40-100cm: [K]
    Soil temperature 100-200cm: [K]
    Soil moisture 0-10cm: [m^3/m^3]
    Soil moisture 10-40cm: [m^3/m^3]
    Soil moisture 40-100cm: [m^3/m^3]
    Soil moisture 100-200cm: [m^3/m^3]

[OUTPUTS]
Baseflow-groundwater runoff [mm/s]: Name of the Baseflow-groundwater runoff downloaded dataset.
Storm surface runoff [mm/s]: Name of the Storm surface runoff downloaded dataset.
Pressure [Pa]: Name of the Pressure downloaded dataset.
Evapotranspiration [mm/s]: Name of the Evapotranspiration downloaded dataset.
Temperature [K]: Name of the Temperature downloaded dataset.
Temperature Min [K]: Name of the Temperature Min downloaded dataset.
Temperature Max [K]: Name of the Temperature Max downloaded dataset.
Total Precipitation Rate [mm/s]: Name of the Total Precipitation Rate downloaded dataset.
Wind speed [m/s]: Name of the Wind speed downloaded dataset.
Soil temperature 0-10cm [K]: Name of the Soil temperature 0-10cm downloaded dataset.
Soil temperature 10-40cm [K]: Name of the Soil temperature 10-40cm downloaded dataset.
Soil temperature 40-100cm [K]: Name of the Soil temperature 40-100cm downloaded dataset.
Soil temperature 100-200cm [K]: Name of the Soil temperature 100-200cm downloaded dataset.
Soil moisture 0-10cm [m^3/m^3]: Name of the Soil moisture 0-10cm downloaded dataset.
Soil moisture 10-40cm [m^3/m^3]: Name of the Soil moisture 10-40cm downloaded dataset.
Soil moisture 40-100cm [m^3/m^3]: Name of Soil moisture 40-100cm downloaded dataset.
Soil moisture 100-200cm [m^3/m^3]: Name of the Soil moisture 100-200cm downloaded dataset.
