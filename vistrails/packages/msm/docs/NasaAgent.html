NasaAgent

 access the Earth Data services of NASA.
dataset: This corresponds to the Nasa dataset as listed here:
http://hydro1.sci.gsfc.nasa.gov:80/dods/\t
Example: NLDAS_FORA0125_H.002\n
empty_value: Representation for missing data. It can be found in the 'info' link in front of each dataset.
Example: https://hydro1.gesdisc.eosdis.nasa.gov/dods/NLDAS_FORA0125_H.002.info\n
For the above dataset, the empty value is: -9.99E8\n
to_meters: Factor used to convert the received values, for instance, into metric system (It can be used for any other unit conversion).
For example, to convert the data at NLDAS_FORA0125_H.002 from kg-water/m^2 to m-water/m^2 'to_meters' = 0.001\n
url: Url given by the OpenDap provider, in the case of Nasa OpenDap:
http://:@hydro1.sci.gsfc.nasa.gov:80/dods/\n
varname_at_src: Name of the variable in the dataset. For the dataset NLDAS_FORA0125_H.002
all of the following are valid variable names: apcpsfc, convfracsfc, dlwrfsfc, etc.
