msmDatasetOperation

 performs a mathematical operation to a dataset.
The operations needs to be specified in terms of x. For example:
the operation 2*x+1 multiplies a dataset by 2 and adds 1. All operations are performed
using the eval() function [https://docs.python.org/3/library/functions.html#eval].

[INPUTS]
dataset_name1: The first dataset you want to operate. Dataset x.
dataset_name2: The second dataset you want to operate. Dataset y.
variable_name: The new variable that will be created by the operation.
(i.e 'distance'/'time' = 'velocity').
operation: The mathematical operation you want to apply (i.e. (y*x+2*y)/x).
units: The resulting units of your operation.

[OUTPUTS]
resulting_dataset_name:The name of the resulting dataset.
