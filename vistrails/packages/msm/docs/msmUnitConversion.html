msmUnitConversion

 performs a mathematical operation to a dataset.
The operations needs to be specified in terms of x. For example:
the operation 2*x+1 multiplies a dataset by 2 and adds 1. All operations are performed
using the eval() function [https://docs.python.org/3/library/functions.html#eval].

[INPUTS]
dataset_name: The dataset you want to operate/change units. Dataset x
operation: The mathematical operation you want to apply (i.e. 2*x+pow(x,2)).
new_units: [Optional] The resulting units of your operation.

[OUTPUTS]
resulting_dataset_name: The name of the transformed dataset.
