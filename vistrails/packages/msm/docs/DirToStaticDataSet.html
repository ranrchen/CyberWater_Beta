DirToStaticDataSet

 Creates a single dataset from a folder, where only one map is loaded .
as a static map. This map will have a date as the current time by default.

[INPUTS]
01_space_range: Space range for the dataset that will be uploaded.
02_variable_name: Name of the variable that will be stored in the dataset. i.e. precipitation, temperature, etc..
(this value is only used as label and does not play a role in the calculations.)
03_dirname: Path where the dataset information is stored..
04_units: [Optional] Units of the dataset that will be uploaded. This value will be used only as a label.
(this value is only used as label and does not play a role in the calculations.)
05_empty_symbol: [Optional] String representation of empty (NULL, None) values.

[OUTPUTS]
out_dataset_name: The name of the resulting dataset.
