msmMetaDataAgent

 module displays the metadata of a given dataset.
The metadata of a certain dataset can be display in a "Metadata Viewer" windows for detail information
User can connect the following modules with corresponding output ports for needed meta,
notable, user can connected with RoutingParameter File module to get the updated space range of the dataset retrieved with NLDASAagent, by connecting it to the "SpaceRange" output port of msmMetaDataAgent module.

[INPUTS]
Dataset: The name of the dataset.

[OUTPUTS]
Base/Horizontal Resolution: The horizontal resolution of the dataset
Side/Vertical Resolution: The vertical resolution of the dataset
Step/Time Delta: The time delta of the dataset
Variable: The variable name of the dataset
Unit: The unit of the dataset
TimeRange: The starting time and the ending time of the dataset, which can be connected with SpaceRange module
SpaceRange: The boundary values for west-end, east-end, north-end, south-end respectively of the dataset, which can be connected with RoutingParameterFile module to pass the updated SpaceRange of NLDASAgent module.
