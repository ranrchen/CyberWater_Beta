TimeRange

 module defines the temporal extent of the users simulation.
All Data Agents will download the information available within the time limits, at whichever
temporal resolution either chosen or available. The user should be aware of such resolution, and
is responsible for changing it if necessary.

timeini (yyyy/mm/dd HH:MM:SS): Starting time of the simulation.
timeend (yyyy/mm/dd HH:MM:SS): Ending time of the simulation.
