GPMAgent

 access the Earth Data services of NASA, through SSW.
It downloads precipitation information in different temporal resolutions.
Its spatial resolution is 1/10 of a degree. It covers the entire world

You need to create an account on Earth Data following the link:
https://urs.earthdata.nasa.gov/

password:Your Earth Data password credentials.
username:Your Earth Data username credentials.
rename_variable: [Optional] Input used if you want to rename the variable name of this dataset.
time_resolution:The time resolution you want. Possible options:
Daily
Half Hourly
