StaticDataSetToFile

 Exports a dataset with a single time-step into a file.
If the dataset has more than 1 timesteps, a warning is issued, but only the first time-step is exported.

[INPUTS]
01_dataset_ids: [Multiple] One or many dataset-names that will be exported
02_directory: Directory where the files will be created.
03_file_name: Name of the exported file.
