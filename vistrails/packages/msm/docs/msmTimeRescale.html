msmTimeRescale

 Changes the time resolution of a dataset.
This module does not modify the original dataset. It creates a new modified dataset instead.

[INPUTS]
original_resolution_type: The time resolution type of the original dataset. Options: [Hourly, Monthly].
resolution_reset:
resolution_type: The time resolution of the new dataset. Options: [Hourly, Daily,Weekly, Monthly, Yearly].
resolution_number: The granularity based on the resolution type of the new dataset, Default value is 1.
e.g., Translate the time resolution of a dataset from Hourly data to Daily data, user can click Daily and leave the text area of resolution_number blank,
or choose the Hourly as the resolution_type and set resolution_number as 24.
resolution_number:The new variable that will be created by the operation.
variable_type: Variable-type of measurement. Options: [accumulated, average].
dataset_name: The dataset that will be used to create the one with different time resolution.
new_variable_name:The new variable that will be created by the operation.

[OUTPUTS]
resampled_dataset_name:The name of the resulting dataset.
