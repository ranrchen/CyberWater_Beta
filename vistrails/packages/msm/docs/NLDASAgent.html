NLDASAgent

 access the Earth Data services of NASA.
NLDASAgent model offers various meteorological variables in an hourly basis, 1/8^o resolution.
You need to create an account on Earth Data following the link:
https://urs.earthdata.nasa.gov/

Projection:Latitude, Longitude
Spatial Coverage: -125.0,25.0,-67.0,53.0
Temporal Coverage: 1979-01-02 to 2020-05-25
Spatial Resolution: 0.125 x 0.125 degrees
Temporal Resolution: Hourly

[INPUTS]
case_study: Tuple of TimeRange-SpaceRange defining the spatio-temporal extent of the download.
The information downloaded will always fall inside these limits, in case the spatial information, or temporal
divisions do not fall exaclty at the limits set by the user in his or her 'studycase'.
password: Your Earth Data password credentials.
username: Your Earth Data username credentials.
convert_to_dataset: Flag on whether to convert the forcing data files to datasets to pass to the successive modules in CyberWater, The value is either True or False.
rename_variable: [Optional] Input used if you want to rename the variables name of this dataset.
variableName: The variables you want to download (including units), which can click multiple variables by clicking the '+' button on the left side.
Possible options:
    Pressure [Pa]
    Potential Evaporation [mm/h]
    Specific Humidity [kg/kg]
    Radiation Flux Long Wave [W/m^2]
    Radiation Flux Short Wave [W/m^2]
    Temperature [K]
    Total Precipitation [mm/h]
    u-wind [m/s]
    v-wind [m/s]

[OUTPUTS]
Pressure [Pa]: Name of the Pressure downloaded dataset
Potential Evaporation [mm/h]: Name of the Potential Evaporation downloaded dataset
Specific Humidity [kg/kg]: Name of the Specific Humidity downloaded dataset
Radiation Flux Long Wave [W/m^2]: Name of the Radiation Flux Long Wave downloaded dataset
Radiation Flux Short Wave [W/m^2]: Name of the Radiation Flux Short Wave downloaded dataset
Temperature [K]: Name of the Temperature downloaded dataset
Total Precipitation [mm/h]: Name of the Total Precipitation downloaded dataset
u-wind [m/s]: Name of the u-wind downloaded dataset
v-wind [m/s]: Name of the v-wind downloaded dataset
Documentation at [https://hydro1.gesdisc.eosdis.nasa.gov/data/NLDAS/README.NLDAS2.pdf]
