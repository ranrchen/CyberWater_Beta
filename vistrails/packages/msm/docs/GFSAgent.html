GFSAgent

 Access the Global Forecast System from NOAA to obtain global data of.
more than 360 variables. 105 major ones are in the list, but the user can search for specific ones.
GFS works with dates available at UTC time.
https://www.ncdc.noaa.gov/data-access/model-data/model-datasets/global-forcast-system-gfs

Projection: Latitude, Longitude
Spatial Coverage: GLOBAL: -180,90,180,-90 (W,N,E,S)
Temporal Coverage: FROM NOW-144 hours TO NOW+9 days
Spatial Resolution: 0.25 x 0.25 degrees
Temporal Resolution: Hourly

[INPUTS]
01_case_study: Tuple of TimeRange-SpaceRange defining the spatio-temporal extent of the download.
grass_engine: Connection to the local GRASS GIS.
reference_date: (UTC) Date at which the requested forecast was created.
delete_files?: Check to delete the local files created.
variable_name: Variable to download. The user can plug a String module with a name that is
not included in the list. The full list of variables is available at
CyberWater\VisTrails\vistrails\packages\msm\utils\uploaded\user_data_agents\GFSAgent\GFS variables.txt
