msmShowChart

 Creates a matplotlib timeseries chart, using the average values of a dataset.
Each point in the time-series is the average of the spatial map of the dataset.

[INPUTS]
dataset_names_mainAxis: All the datasets that the user wants to plot in the left axis.
dataset_names_secondaryAxis: All the datasets that the user wants to plot in the right axis.
units:[Optional] The resulting units of your operation.
scale_font:[Optional] Factor to scale the size of the fonts in the chart: Legend, Axis values, and Axis Labels.
variable_name:[Optional] The name at the left axis ()recommended when only 1 axis is used.
line_type:[Optional] '[marker][line][color]' The pattern used to draw the timeseries.
Some examples: [marker]:('.', ',', '*', '^', 'D'),[line]:('-', '--', ':'),[color]:(b,g,r,k,y,c).
For more details, see the Notes at the bottom of [https://matplotlib.org/3.2.1/api/_as_gen/
matplotlib.pyplot.plot.html#matplotlib.pyplot.plot].
vertical_range:[Optional] The new limits of the vertical axis of the plot, separated by a comma. Format: min,max.

[OUTPUTS]
No outputs.
