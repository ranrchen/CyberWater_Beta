TableParameterAgent creates a table parameter file, taking information from one or more tables.
All parameter entries inputs of TableParameterAgent are created by this module

Parameter_Value: Values to add to the parameter file. It can be either a python function, the path of a MSM ASCII map (which would require to specify the limits), an individual numeric constant, or a list of numeric values.
Parameter_Type: FUNCTION, MAP, NUMERIC, LIST.
Parameter_Position: Sets the column index where the values will be saved in the parameter file If the Entry is a MAP, the user can either set 1 value: the Z in the map. Or 3 values: \longitude, latitude, Z\ in this order.
Map_limits: Coordinate limits. Only necessary if the entry is a MAP. Format: \north, west, south, east\.
