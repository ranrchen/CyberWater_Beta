ColumnEntry

 Prepares the inputs for the a table parameter file.
This module needs to be used in conjunction with the TableParameterAgent module.
All parameter entries inputs of TableParameterAgent are created by this module

Column_Value: Values to add to the table parameter file. Values can come from ither a python function, the path of a ASCII table (which would require to specify the columnIndexes), an individual numeric constant, or a list of numeric constant values.
Column_Type: FUNCTION, TABLE, NUMERIC, LIST.
Column_Position: Sets the column index where the values will be saved in the parameter file.
If the Entry is a TABLE, the user can set 1 or list as many values as specified on Table_ColumnIndex.
Table_ColumnIndex: Index or indexes extracted from the source TABLE. Only required when Column_Type is Table.
Operation: [Optional] Operation that the values \x\ will go. i.e (2*x+1) will double the values and add 1.
