RoutingParameterFile

 creates a table parameter file, taking information from one or more tables.
All parameter entries inputs of TableParameterAgent are created by this module

space_range: Output of an SpaceRange component. This should be the same as the one used for the forcing datasets.
grass_engine: Output of the GrassEngine component.
resolution: resolution used to generate the draining network.
outlet: (longitude,latitude) coordinates of the outlet of the basin in the DEM.
directory: Folder where the routing parameter file will be created.
DEM_path: Path of the file that contains the DEM data in ASCII format.
DEM_limits: (north, west, south, east).
x: Muskingum wedge storage.
k: Muskingum time of travel between the inflow and outflow [h].

[OUTPUTS]
filePath: full path of the routing file created.
