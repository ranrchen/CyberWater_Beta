StaticBinaryMapAgent Creates a binary file, using the ASCII map given as input.
This module requires the user to have GRASS GIS on your local machine.
space_range: A space range defining east, west, north, south limits of the map.
directory: The folder where the binary map will be created.
grass_engine: A Grass-Engine object created with the GISEngine module.
map_path: The path were ASCII map is located in the local machine.
bytes: Bytes per cell of the output file: [1, 2, 4, 8].
resolution(m): Resolution of the resulting map in meter. Default uses the input map resolutions.
