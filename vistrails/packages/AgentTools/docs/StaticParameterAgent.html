StaticParameterAgent

 Creates a parameter file where each row describes a single cell.
All parameter entries inputs of PrepareParameterFile are created by this module

\ all_parameters: All the values that will be cinluded in the file. It is a list of ParameterEntry modules.
\ directory: Folder where the parameter file will be created.
\ fileName: Name of the parameter file.
separator: [Optional] Character separator used in the parameter file generated. Default =\tab.

[OUTPUTS]
filePath: Path of the output file generated.
