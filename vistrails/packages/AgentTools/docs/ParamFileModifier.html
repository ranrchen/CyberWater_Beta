ParamFileModifier

 Changes part of a text file.
The user needs to pick one of the two modes of changing the file.

\ file_path: Path of the file to change.
modify_line_after: Prefix of the line to be modified
modify_line_before: Suffix of the line to me modified
new_text: New text that will be replaced in the file
create_new_file?: Requests whether or not a new file should be created, to prevent the original to be altered.
new_file_name: If a new file is going to be created, this will providde the new name for it.
