ParameterEntry

 Prepares the inputs for the parameter files.
This module needs to be used in conjunction with the StaticParameterAgent module.
All parameter entries inputs of StaticParameterAgent are created by this module

Parameter_Value: Values to add to the parameter file.
It can be either a python function,
the path of a MSM ASCII map (which would require to specify the limits),
an individual numeric constant, or
a list of numeric values.
Parameter_Type: FUNCTION, MAP, NUMERIC, LIST.
Operation: Operation applied to the incoming data (i.e x*0.5 will compute half of the input values).Implemented with the 'Eval()' python function.
Parameter_format: Python format of the values (i.e %.3f is a float with 3 numbers after the point).See [https://docs.python.org/2.4/lib/typesseq-strings.html] for more documentation.
Parameter_Position: Sets the column index where the values will be saved in the parameter file
If the Entry is a MAP, the user can either set 1 value: the Z in the map. Or 3 values:longitude, latitude, Z in this order.
Map_limits: Coordinate limits. Only necessary if the entry is a MAP. Format:north, west, south, east.
Operation: [Optional] Operation that the values \x\ will go. i.e (2*x+1) will double the values and add 1.
