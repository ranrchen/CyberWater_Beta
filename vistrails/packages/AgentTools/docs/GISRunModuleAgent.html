GISRunModuleAgent

 This module executes any GRASS GIS command.
For a full documentation of GRASS GIS commands, visit:.
https://grass.osgeo.org/grass79/manuals.

\ Command_name: Name of the GRASS GIS command.
Parameter: Parameters or arguments of the command, separated by a semicolon ';'.
ReadyToStarti: Flag to indicate if this module should start or not.
grass_enginei: The conection to GRASS GIS from the GISEngine module.

[OUTPUTS]
Ready: Ready signal. If it fails, it sends a FALSE.
