VICVegetationParamAgent

 Capable of creating a Vegetation Parameter File for VIC.
It offers three different modes to create the vegetation parameter file:
  1. Default with user-custom LAI: The veg_param file will contain the same vegetation types (2, 6, 7, 10, 11)in every
    cell with the depths and roots input as parameters. The LAI used comes
    from the LAI_cycle parameter. Required parameters: dem_map_name,depths,roots
    LAI_cycle, directory, parameter_file_name    .

  2. Full computation: The vegetation cover of every cell in the map is computed.This process takes a couple
    of seconds depending on the resolution of the vegetation map used.The LAI assgined to
    each vegetation type is extracted from the vegetation library file. Please note that
    both the veg-lib file and the vegetation map must share the same legend. The same
    depths and roots are used for every cell. Required parameters: dem_map_name, depths,
    roots, veg_lib_file, veg_map_file, veg_map_limits, GIS_engine, directory, parameter_file_name.

  3. Update LAI only mode: Takes an already-existing vegetation parameter file and only updates the LAI cycles of
    each vegetation type, from the vegetation library file. Required parameters:
    only_update_LAI(TRUE), veg_lib_file, directory, parameter_file_name.
The default legend for the vegetation types is taken from the legend used by the Multi-Consortium
Land Characteristics (MRLC) at [https://www.mrlc.gov/data/legends/national-land-cover-database-2016-nlcd2016-legend]

VEGETATION LEGEND:
  1. Water
  2. Urban - Open space
  3. Urban - Low intensity
  4. Urban - Medium intensity
  5. Urban - High intensity
  6. Barren Land
  7. Deciduous forest
  8. Evergreen forest
  9. Mixed forest
  10. Shrubs
  11. Grassland
  12. Pasture/Hay
  13. Cultivated crops
  14. Woody Wetlands
  15. Emergent Herbaceous Wetlands
VEGETATION LIBRARY Docs: https://vic.readthedocs.io/en/master/Documentation/Drivers/Classic/VegLib/
VEGETATION PARAMET Docs: https://vic.readthedocs.io/en/master/Documentation/Drivers/Classic/VegParam/
< br > dem_map_name: Path of the DEM map used to know how many cells the VIC model has. [Modes: 1 (default) and 2 (full)].
depths: Depths of each one of the soil layers. Format: \D_1,D_2,...,D_n\. [Modes: 1 (default) and 2 (full)]
directory: Directory where the Vegetation parameter file will be created.
only_update_LAI: Whether or not the parameter file will only have its LAI values updated. [Mode: 3(update LAI)]
parameter_file_name: Name of the Vegetation parameter file.
roots: Percentage of roots on each one of the soil layers. Format: \r_1,r_2,...,r_n\. [Modes: 1 (default) and 2 (full)]
veg_lib_file: Path of the VIC Vegetation library file. [Modes: 2 (full) and 3 (update LAI)]
veg_map_file: Path of the vegetation file. [Mode: 2 (full)]
veg_map_limits: Coordinate limits. Only necessary Veg-map is given. Format: \north,west,south,east\. [Mode: 2 (full)]
LAI_cycle: 12-months LAI cycle. Only necessary when Veg-map is not given. Format: \JAN,FEB,MAR,...\. [Mode: 1 (default)]
GIS_engine: GRASS GISEngine ready to perform GIS operations. [Mode: 2 (full)]
