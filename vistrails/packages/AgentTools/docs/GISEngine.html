GISEngine

 This module simply creates a single GRAS_GIS_Engine object.
All modules that require the use of GIS functions will require a GISEngine

[INPUTS]
grass_bat_file: [Optional] Path to the .bat file to execute GRASS GIS. This is only used when
GRASS GIS is not installed in the default path on your Operative System C:\Program Files\GRASS GIS 7.2.0\grass72.bat
grass_location: [Optional] Name of the GRASS working location. Default = "demolocation"
grass_mapset: [Optional] Name of the GRASS working mapset. Default = "PERMANENT".

[OUTPUTS]
GRASS_GISEngine: A connection to the GIS Engine of GRASS GIS.
